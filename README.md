# genome-insider
- Genome Insider is a collection of explanations, notes, and explorations at the intersection of biology, computation, and medicine.
- Created by Tanaya Datar, this platform explores bioinformatics, molecular biology, immunology, and computational approaches to understanding complex biological systems.
- Genome Insider was designed and built by me using Astro and is deployed with Vercel. The goal was to create a clean, fast, and distraction-free space for exploring biology, genomics, and computational science.
- View the website here: https://genome-insider.vercel.app/
